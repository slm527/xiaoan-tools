## 📝 使用说明

1. 需求分析阶段 → 用 `01` 调研表+报告模板
2. 立项申报阶段 → 用 `02` 项目建议书模板
3. 可研审批阶段 → 用 `03` 可研模板（55号文/57号文）
4. 设计阶段 → 用 `04` 初设方案模板
5. 报价阶段 → 用 `05` 功能点法测算（参考各地标准）
